# Live Demo Flow

## Objective
Demonstrate that Access Assurance Hub can unify access review, policy flags, and audit evidence in one product flow.

## Steps

1. Open Dashboard
   - show active campaigns
   - show pending reviews
   - explain this is the manager landing page

2. Open Inbox
   - point out SAP and AD review items
   - highlight severity and issue context

3. Open John Smith review
   - show assignments
   - explain SoD conflict
   - note that the UI is presentation only

4. Submit decision
   - choose "Revoke risk items" or "Flag follow-up"
   - mention that backend owns workflow
   - mention that OPA evaluated policy

5. Open Audit view
   - show timeline entry for submitted decision
   - show evidence item generated

6. Close with architecture story
   - thin UI
   - backend orchestration
   - OPA sidecar
   - future trust ledger / VC / OSCAL path
